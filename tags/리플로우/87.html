[TECH-QA] 리베이스 (rebase 중 충돌)

Posted 2 months Updated 2 months
By 베짱이가 되고싶은 개미。

📝 rebase 범위 잡기

git log --oneline --grep="^chore [AWS-1280]"

//커밋내역				
a1b2c3d chore: [AWS-1280] typo
e4f5g6h chore: [AWS-1280] notification ui
i7j8k9l chore: [AWS-1280] project

📝 최근 n개의 커밋을 인터랙티브하게 수정

git rebase -i HEAD~n

📝 에디터로 커밋내역을 열어 수정할 커밋앞의 키워드를 바꿔 준다.

pick → reword
reword a1b2c3d fix typo
pick e4f5g6h add notification ui
pick i7j8k9l init project
원격에 반영 (해시 변경됨)
git push origin 브랜치명 --force-with-lease
