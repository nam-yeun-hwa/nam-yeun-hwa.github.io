HTML(하이퍼텍스트 마크업 언어)에서 속성(Attribute)과 프로퍼티(Property)

Posted a year Updated a year
By 베짱이가 되고싶은 개미。
HTML(하이퍼텍스트 마크업 언어)에서 속성(Attribute)과 프로퍼티(Property)는 특정 요소에 대한 정보를 지정하는 데 사용되는 용어입니다. 그러나 두 용어 간에는 몇 가지 차이가 있습니다.

속성(Attribute)

  • HTML 요소에 대한 부가적인 정보를 지정하는 데 사용됩니다.
  • 이름과 값의 쌍으로 구성됩니다.
예를 들면 다음과 같습니다.
<a href="https://www.example.com">링크</a>
위의 예에서 href는 속성이며, "https://www.example.com"은 해당 속성의 값입니다.

프로퍼티(Property)

  • JavaScript에서 DOM(문서 객체 모델)을 통해 HTML 요소에 접근할 때 사용되는 용어입니다.
  • HTML 요소의 속성은 대응하는 DOM 객체의 프로퍼티로 매핑됩니다.
예를 들면 다음과 같습니다.
<div id="myDiv">내 디브</div>

var element = document.getElementById("myDiv");
console.log(element.id); // "myDiv"
간단히 말하면, 속성은 HTML에서 정의되어 있고 문서를 렌더링할 때 사용되며, 프로퍼티는 JavaScript를 통해 DOM을 조작할 때 사용됩니다. 일반적으로 두 용어가 상호 운용되기 때문에 많은 경우에는 두 용어를 혼용하여 사용하는 것이 일반적입니다.
