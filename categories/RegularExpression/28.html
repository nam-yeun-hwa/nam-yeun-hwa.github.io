TailwindлҠ” мң нӢёлҰ¬нӢ° нҚјмҠӨнҠё(Utility-first) CSS н”„л Ҳмһ„мӣҢнҒ¬мһ…лӢҲлӢӨ. мқҙлҠ” кё°ліём ҒмңјлЎң лҜёлҰ¬ м •мқҳлҗң нҒҙлһҳмҠӨлҘј нҷңмҡ©н•ҙ л№ лҘҙкІҢ мҠӨнғҖмқјл§Ғн•  мҲҳ мһҲлҸ„лЎқ лҸ„мҷҖмЈјлҠ” н”„л Ҳмһ„мӣҢнҒ¬лҘј мқҳлҜён•©лӢҲлӢӨ. кё°мЎҙмқҳ CSS л°©мӢқкіјлҠ” лӢӨлҘҙкІҢ TailwindлҠ” лі„лҸ„мқҳ мҠӨнғҖмқј мӢңнҠёлҘј мһ‘м„ұн•ҳлҠ” лҢҖмӢ , HTML нҒҙлһҳмҠӨм—җ м§Ғм ‘ мҠӨнғҖмқјл§Ғ кҙҖл Ё мң нӢёлҰ¬нӢ° нҒҙлһҳмҠӨл“Өмқ„ м¶”к°Җн•ҳлҠ” л°©мӢқмңјлЎң мһ‘м—…н•©лӢҲлӢӨ.

мң нӢёлҰ¬нӢ° нҒҙлһҳмҠӨ

TailwindлҠ” к°Ғ мҠӨнғҖмқјмқ„ лӢЁмқј нҒҙлһҳмҠӨм—җ мәЎмҠҗнҷ”н•©лӢҲлӢӨ.
мҳҲлҘј л“Өм–ҙ, text-center, bg-blue-500, p-4мҷҖ к°ҷмқҖ нҒҙлһҳмҠӨлҘј мЎ°н•©н•ҙ н…ҚмҠӨнҠё м •л ¬, л°°кІҪмғү м„Өм •, нҢЁл”© л“ұмқ„ к°„лӢЁн•ҳкІҢ м Ғмҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

л°ҳмқ‘нҳ• л””мһҗмқё

TailwindлҠ” кё°ліём ҒмңјлЎң л°ҳмқ‘нҳ• л””мһҗмқёмқ„ мүҪкІҢ мІҳлҰ¬н•  мҲҳ мһҲлҸ„лЎқ лҸ„мҷҖмӨҚлӢҲлӢӨ.
мҳҲлҘј л“Өм–ҙ, md:text-centerмҷҖ к°ҷмқҙ нҠ№м • нҷ”л©ҙ нҒ¬кё°м—җм„ңл§Ң м Ғмҡ©лҗ  мҠӨнғҖмқјмқ„ м •мқҳн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

лҜёлҰ¬ м •мқҳлҗң н…Ңл§Ҳ

TailwindлҠ” мғүмғҒ, кёҖкјҙ нҒ¬кё°, м—¬л°ұ л“ұмқ„ лҜёлҰ¬ м •мқҳлҗң к°’мңјлЎң м ңкіөн•ҳл©°, мқҙлҘј мүҪкІҢ м»ӨмҠӨн„°л§Ҳмқҙм§•н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

мң м—°м„ұ

TailwindлҠ” л§Өмҡ° мң м—°н•ҳм—¬ н”„лЎңм қнҠёмқҳ мҠӨнғҖмқј к°Җмқҙл“ңлҘј л”°лҘҙл©ҙм„ңлҸ„ м„ёл¶Җм ҒмңјлЎң лӢӨм–‘н•ң мҠӨнғҖмқјмқ„ м Ғмҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

JIT(Just-In-Time)

н•„мҡ”н•  л•Ңл§ҲлӢӨ мҠӨнғҖмқјмқ„ мғқм„ұн•ҙ мЈјлҠ” JIT лӘЁл“ңлҘј м§Җмӣҗн•ҳм—¬, мӮ¬мҡ©н•ҳм§Җ м•ҠлҠ” CSS нҒҙлһҳмҠӨк°Җ л№Ңл“ңм—җ нҸ¬н•Ёлҗҳм§Җ м•ҠлҸ„лЎқ н•©лӢҲлӢӨ. мқҙлЎң мқён•ҙ л№Ңл“ң нҒ¬кё°лҘј нҒ¬кІҢ мӨ„мқј мҲҳ мһҲмҠөлӢҲлӢӨ.
