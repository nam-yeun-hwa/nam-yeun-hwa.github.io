괄호문자제거

Posted Aug 8, 2019Updated Sep 23, 2023
By Carys Chung
문제 설명
입력된 문자열에서 소괄호 ( ) 사이에 존재하는 모든 문자를 제거하고 남은 문자만 출력하는 프로그램을 작성하세요.
제한 사항
첫 줄에 문자열이 주어진다. 문자열의 길이는 100을 넘지 않는다.
남은 문자만 출력한다.
입출력 예
s result
(A(BC)D)EF(G(H)(IJ)K)LM(N) EFLM 
function solution(s){  
      let answer;
      let stack=[];
      for(let x of s){
          if(x===')'){
          //★★★★★★ 
              while(stack.pop()!=='(');
          }
          else stack.push(x);
      }
      answer=stack.join('');
      return answer;
  }
