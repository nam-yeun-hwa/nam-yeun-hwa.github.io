후위식 연산(postfix)

Posted 3 months Updated 3 months
By 베짱이가 되고싶은 개미。
문제 설명
후위연산식이 주어지면 연산한 결과를 출력하는 프로그램을 작성하세요.
만약 3(5+2)-9 을 후위연산식으로 표현하면 352+9- 로 표현되며 그 결과는 12입니다.
제한 사항
첫 줄에 후위연산식이 주어집니다.
연산식의 길이는 50을 넘지 않습니다. 식은 1~9의 숫자와 +, -, *, / 연산자로만 이루어진다.
입출력 예
str result
352+*9- 12 
function solution(s){  
      let answer;
      let stack=[];
      for(let x of s){
          if(!isNaN(x)) stack.push(Number(x));
          else{
              let rt=stack.pop();
              let lt=stack.pop();
              if(x==='+') stack.push(lt+rt);
              else if(x==='-') stack.push(lt-rt);
              else if(x==='*') stack.push(lt*rt);
              else if(x==='/') stack.push(lt/rt);
          }
      }
      answer=stack[0];
      return answer;
  }
  
  let str="352+*9-";
  console.log(solution(str));
